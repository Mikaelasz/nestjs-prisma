## 📚 Aplicação de Livros 📖

### Descrição

Este é um projeto de uma aplicação de livros, desenvolvida utilizando Nest.js e Prisma. A aplicação permite a criação, listagem, alteração e remoção de livros em um banco de dados.

### Funcionalidades

- 📝 Criação de livros
- 📋 Listagem de livros
- ✏️ Alteração de livros
- 🗑️ Remoção de livros
